This project implements an AI-based fault detection system using signal simulation in MATLAB and machine learning in Python.

---

##  Features

- Synthetic signal generation (normal & faulty signals)
- Feature extraction (mean, variance, RMS, skewness, kurtosis)
- Fault classification using Support Vector Machine (SVM)
- Model evaluation using accuracy and confusion matrix

---

##  Technologies Used

- MATLAB (signal generation & feature extraction)
- Python (scikit-learn for ML model)

---

##  Results

- Achieved high classification accuracy
- Clear separation between normal and faulty signals

---

## Project Workflow

1. Generate signals in MATLAB  
2. Extract features  
3. Export dataset to CSV  
4. Train and test model in Python
